Android:
1. Very popular.
2. Development can be done in Linux/Windows.
3. Has a linux shell, which I am conversant with
4. Gives raw data as output
5. Free software
6. Programming in Java. No experience. But seems close to C++. Android NDK can be coded in C++.
7. Bluetooth can be used for data transmission.

Windows phone 8:
1. New platform
2. Development only in Windows 8(May need to use my computer or install windows 8 in one of the systems).
3. Freeware
4. Gives raw data as well as polished data
5. Programming in C#. No stark differences between C# and C++. I am conversant with C++.
6. Not sure about USB but Socket communication is possible.
