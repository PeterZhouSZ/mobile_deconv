This document contains notes regarding the windows phone 8 sdk

Installation:
1. Download only the full installer from dream spark. Though visual studio express is free, download from dreamspark will ensure that you can also deploy the application on the mobile without paying 99$ fee.

Important instructions:
1. If ACCESS_DENIED error pops up, permissions can be changed in WMAppManifest.xml file.
2. A delegate() is like a function pointer.
3. Accessing UI elements directly is not allowed. A call should be done using this.Dispatcher.BeginInvoke(action)
